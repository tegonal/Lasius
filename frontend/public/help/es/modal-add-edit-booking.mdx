# Añadir/Editar Reserva

Crea nuevas reservas o modifica las existentes para realizar un seguimiento preciso de tu tiempo en los proyectos.

## Inicio Rápido con Preajustes

Haz clic en **Explorar preajustes** en la parte superior para seleccionar rápidamente entre:
- **Recientes**: Tus combinaciones de proyecto y etiqueta utilizadas recientemente
- **Favoritos**: Reservas que has marcado como favoritas para acceso rápido
- **Equipo**: Reservas populares de los miembros de tu equipo

Al seleccionar un preajuste se rellenan automáticamente el proyecto y las etiquetas.

## Detalles de la Reserva

### Proyecto y Etiquetas
1. Selecciona un **Proyecto** del menú desplegable (obligatorio)
2. Añade **Etiquetas** para categorizar tu trabajo
   - Escribe para buscar o selecciona entre las etiquetas disponibles
   - Usa grupos de etiquetas para una categorización coherente

### Entrada de Tiempo

#### Campos de Fechas y Hora
Cada campo de tiempo admite entrada por teclado para una introducción rápida:
- **Escribe directamente**: Introduce fechas con formatos cortos que se expanden automáticamente
- **Navega entre segmentos**: Usa Tab o las flechas para moverse entre día/mes/año y hora/minuto
- **Selecciona y ajusta**: Haz clic en un segmento (día, mes, año, hora, minuto) para seleccionarlo, luego usa las flechas o haz clic en los botones arriba/abajo para cambiar su valor
- **Entrada rápida**: Escribe `:` o `.` para saltar al siguiente segmento
- **Restablecer**: Pulsa Escape para volver al valor inicial

#### Horas de Inicio y Fin
- **Inicio**: Cuándo comenzó el trabajo
- **Fin**: Cuándo terminó el trabajo
- Haz clic en los iconos de preajuste (<Icon name="ArrowDownToLine" /> <Icon name="ArrowUpToLine" />) para:
  - Usar la hora de fin de la reserva anterior/más reciente como hora de inicio
  - Usar la hora de inicio de la siguiente reserva como hora de fin

#### Duración
- Muestra el tiempo calculado entre inicio y fin
- Edita directamente para ajustar la hora de fin
- Utiliza los mismos atajos de teclado que los campos de tiempo
- Admite incrementos de 5 minutos al hacer clic en las flechas sin seleccionar un segmento

## Consejos

<Tip>Usa atajos de teclado para introducir horas rápidamente: escribe números, usa flechas para ajustar, pulsa Tab para moverte entre campos</Tip>

<Tip>Explora los preajustes para ahorrar tiempo al crear reservas similares</Tip>

<Tip>El campo de duración se calcula automáticamente desde tus horas de inicio y fin</Tip>

<Tip>Pulsa Escape en cualquier campo de tiempo para deshacer los cambios y volver al valor original</Tip>
