# Añadir/Editar Reserva

Cree nuevas reservas o modifique las existentes para realizar un seguimiento preciso de su tiempo en los proyectos.

## Inicio Rápido con Preajustes

Haga clic en **Explorar preajustes** en la parte superior para seleccionar rápidamente entre:
- **Recientes**: Sus combinaciones de proyecto y etiqueta utilizadas recientemente
- **Favoritos**: Reservas que ha marcado como favoritas para acceso rápido
- **Equipo**: Reservas populares de los miembros de su equipo

Al seleccionar un preajuste se rellenan automáticamente el proyecto y las etiquetas.

## Detalles de la Reserva

### Proyecto y Etiquetas
1. Seleccione un **Proyecto** del menú desplegable (obligatorio)
2. Añada **Etiquetas** para categorizar su trabajo
   - Escriba para buscar o seleccione entre las etiquetas disponibles
   - Use grupos de etiquetas para una categorización coherente

### Entrada de Tiempo

#### Campos de Fecha y Hora
Cada campo de tiempo admite entrada por teclado para una introducción rápida:
- **Escriba directamente**: Introduzca fechas con formatos cortos que se expanden automáticamente
- **Navegue entre segmentos**: Use Tab o las flechas para moverse entre día/mes/año y hora/minuto
- **Seleccione y ajuste**: Haga clic en un segmento (día, mes, año, hora, minuto) para seleccionarlo, luego use las flechas o haga clic en los botones arriba/abajo para cambiar su valor
- **Entrada rápida**: Escriba `:` o `.` para saltar al siguiente segmento
- **Restablecer**: Pulse Escape para volver al valor inicial

#### Horas de Inicio y Fin
- **Inicio**: Cuándo comenzó el trabajo
- **Fin**: Cuándo terminó el trabajo
- Haga clic en los iconos de preajuste (<Icon name="ArrowDownToLine" /> <Icon name="ArrowUpToLine" />) para:
  - Usar la hora de fin de la reserva anterior/más reciente como hora de inicio
  - Usar la hora de inicio de la siguiente reserva como hora de fin

#### Duración
- Muestra el tiempo calculado entre inicio y fin
- Edite directamente para ajustar la hora de fin
- Utiliza los mismos atajos de teclado que los campos de tiempo
- Admite incrementos de 5 minutos al hacer clic en las flechas sin seleccionar un segmento

## Consejos

<Tip>Use atajos de teclado para introducir horas rápidamente: escriba números, use flechas para ajustar, pulse Tab para moverse entre campos</Tip>

<Tip>Explore los preajustes para ahorrar tiempo al crear reservas similares</Tip>

<Tip>El campo de duración se calcula automáticamente desde sus horas de inicio y fin</Tip>

<Tip>Pulse Escape en cualquier campo de tiempo para deshacer los cambios y volver al valor original</Tip>
