# Historial de Reservas

Vea, filtre y exporte todas sus reservas de tiempo en formato de tabla.

## Qué Puede Hacer

### Ver Reservas
Su historial de reservas se muestra en una tabla que muestra proyecto, etiquetas, fecha/hora y duración.

### Filtrar Reservas
Use el panel de filtros a la derecha para reducir su vista:
- Seleccione proyectos específicos
- Filtre por etiquetas
- Elija un rango de tiempo (hoy, esta semana, este mes o fechas personalizadas)
- Haga clic en **Restablecer** para limpiar todos los filtros

### Hacer Clic en Etiquetas
Haga clic en cualquier chip de etiqueta en la tabla para filtrar instantáneamente las reservas por esa etiqueta.

### Acciones de Reserva
Haga clic en el menú contextual (<Icon name="EllipsisVertical" />) en cualquier reserva para:
- **Iniciar reserva** - Comenzar a realizar un seguimiento del tiempo con el mismo proyecto y etiquetas
- **Editar reserva** - Cambiar hora de inicio/fin, proyecto o etiquetas
- **Ajustar inicio a reserva anterior** - Alinear la hora de inicio con el fin de la reserva anterior
- **Añadir como favorito** - Guardar este proyecto para acceso rápido
- **Eliminar reserva** - Eliminar esta entrada

### Exportar Datos
Haga clic en el botón **Exportar** en la parte superior para descargar sus reservas como archivos CSV o Excel.
