# Historial de Reservas

Ve, filtra y exporta todas tus reservas de tiempo en formato de tabla.

## Qué Puedes Hacer

### Ver Reservas
Tu historial de reservas se muestra en una tabla que muestra proyecto, etiquetas, fecha/hora y duración.

### Filtrar Reservas
Usa el panel de filtros a la derechas para reducir tu vista:
- Selecciona proyectos específicos
- Filtra por etiquetas
- Elige un rango de tiempo (hoy, esta semana, este mes o fechas personalizadas)
- Haz clic en **Restablecer** para limpiar todos los filtros

### Hacer Clic en Etiquetas
Haz clic en cualquier chip de etiqueta en la tabla para filtrar instantáneamente las reservas por esa etiqueta.

### Acciones de Reserva
Haz clic en el menú contextual (<Icon name="EllipsisVertical" />) en cualquier reserva para:
- **Iniciar reserva** - Comenzar a realizar un seguimiento del tiempo con el mismo proyecto y etiquetas
- **Editar reserva** - Cambiar hora de inicio/fin, proyecto o etiquetas
- **Ajustar inicio a reserva anterior** - Alinear la hora de inicio con el fin de la reserva anterior
- **Añadir como favorito** - Guardar este proyecto para acceso rápido
- **Eliminar reserva** - Eliminar esta entrada

### Exportar Datos
Haz clic en el botón **Exportar** en la parte superior para descargar tus reservas como archivos CSV o Excel.
