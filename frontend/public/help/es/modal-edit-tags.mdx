# Editar Etiquetas del Proyecto

Gestione las etiquetas y grupos de etiquetas de su proyecto para categorizar y filtrar reservas para mejores informes.

## Qué Puede Hacer

### Crear Grupos de Etiquetas
Los grupos de etiquetas agrupan múltiples etiquetas para aplicarlas rápidamente durante la reserva:
1. Introduzca un nombre en el campo **Nombre**
2. Haga clic en **Crear grupo de etiquetas**
3. El nuevo grupo aparece debajo con un campo de entrada
4. Escriba nombres de etiquetas y pulse Enter para añadir etiquetas al grupo
5. Haga clic en el icono de eliminar (<Icon name="Trash2" />) para eliminar el grupo completo

### Añadir Etiquetas Simples
Añada etiquetas individuales que se pueden usar de forma independiente:
1. Haga clic en el campo **Introduzca una etiqueta para añadirla**
2. Escriba el nombre de la etiqueta y pulse Enter
3. La etiqueta aparece encima del campo de entrada
4. Haga clic en el icono de quitar (<Icon name="X" />) en cualquier etiqueta para eliminarla

### Añadir Grupos de Etiquetas Predeterminados
Haga clic en **Añadir grupos de etiquetas predeterminados** para crear automáticamente un conjunto estándar de grupos de etiquetas incluyendo estado de facturación y tipos de trabajo.

### Guardar Cambios
Haga clic en **Guardar** para aplicar sus cambios de etiquetas al proyecto, o **Cancelar** para descartar los cambios.

## Consejos

<Tip>Los grupos de etiquetas aseguran un etiquetado coherente entre los miembros del equipo</Tip>

<Tip>Use etiquetas simples para categorías puntuales que no necesitan agruparse</Tip>

<Tip>Las etiquetas permiten un filtrado detallado en las vistas de Estadísticas y Listas</Tip>
