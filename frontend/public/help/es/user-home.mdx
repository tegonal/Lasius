# Reservas - Página de Inicio

Realiza un seguimiento de tu tiempo y gestiona tus reservas diarias.

## Qué Puedes Hacer

### Navegar Entre Días
Usa el selector de fechas en la parte superior para ver las reservas de diferentes días. Haz clic en las flechas para navegar o haz clic en un botón de día específico. Cada botón de día muestra una barra de progreso que indica cuánto de tu jornada laboral esperada has registrado.

### Cambiar de Organización
Usa el selector de organización en la barra superior si eres miembro de múltiples equipos.

### Ver Reserva Actual
Al realizar un seguimiento del tiempo, la barra de estado muestra el nombre del proyecto activo, la duración en curso y un botón de detener (<Icon name="Square" />) para finalizar el seguimiento.

### Gestionar las Reservas de Hoy
Cada entrada de reserva muestra proyecto, etiquetas, hora de inicio/fin y duración. Entre las reservas, las flechas indicadoras muestran espacios o superposiciones en tu seguimiento de tiempo.

Haz clic en el menú contextual (<Icon name="EllipsisVertical" />) en cualquier reserva para:
- **Iniciar reserva** - Comenzar a realizar un seguimiento del tiempo con el mismo proyecto y etiquetas
- **Editar reserva** - Modificar hora de inicio, hora de fin, proyecto o etiquetas
- **Ajustar inicio a reserva anterior** - Alinear automáticamente la hora de inicio con el fin de la reserva anterior
- **Añadir como favorito** - Marcar este proyecto para acceso rápido
- **Eliminar reserva** - Eliminar esta entrada de tiempo

### Insertar Reservas
Usa los botones **Añadir reserva** e **Insertar reserva** entre entradas para rellenar espacios en tu cronología.

### Comenzar a Realizar Seguimiento del Tiempo
El panel derecho ofrece tres formas de comenzar el seguimiento:

**Pestaña 1: Iniciar Reserva** (<Icon name="Clock" />)
- Selecciona un proyecto del menú desplegable (obligatorio)
- Añade etiquetas opcionales para categorizar tu trabajo
- Haz clic en **Iniciar reserva** (<Icon name="PlayCircle" />) para comenzar el seguimiento inmediatamente
- O haz clic en **Crear una reserva** para añadir una entrada manual con horas específicas

**Pestaña 2: Iniciar desde Favorito** (<Icon name="Star" />)
- Haz clic en cualquier reserva favorita para comenzar rápidamente a realizar el seguimiento con esa combinación de proyecto y etiqueta
- Usa el menú contextual (<Icon name="EllipsisVertical" />) para acciones adicionales

**Pestaña 3: Iniciar desde Miembro del Equipo** (<Icon name="Usars" />)
- Haz clic en el avatar de un miembro del equipo para ver tus reservas recientes
- Copia tus reservas para realizar el seguimiento de los mismos proyectos

## Consejos

<Tip>El selector de fechas en la parte superior te permite revisar y editar reservas pasadas</Tip>

<Tip>Usa etiquetas de forma coherente para generar informes significativos</Tip>

<Tip>El menú contextual (<Icon name="EllipsisVertical" />) en cada reserva ofrece acciones rápidas</Tip>

<Tip>Usa "Añadir como favorito" en el menú contextual para guardar combinaciones de proyecto y etiqueta de uso frecuente para acceso rápido</Tip>

<Tip>La pestaña "Iniciar desde miembro del equipo" facilita ver en qué están trabajando tus compañeros</Tip>
