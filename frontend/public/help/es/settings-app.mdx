# Configuración de la aplicación

Personaliza tu experiencia Lasius configurando las preferencias de la interfaz.

## Lo que puedes hacer

### Elige tu idioma
Selecciona tu idioma de interfaz preferido del menú desplegable

### Selecciona tu tema
Elige un tema que se adapte a tus preferencias

### Controla el tutorial de bienvenida
Activa o desactiva el tutorial de bienvenida

Haz clic en **Guardar configuración** para aplicar todos tus cambios. La página se recargará para activar tus nuevas preferencias.

## Consejos

<Tip>Cambiar el idioma recargará la aplicación para aplicar las traducciones en todas partes</Tip>

<Tip>El tema del sistema cambia automáticamente entre claro y oscuro según la configuración de tu dispositivo</Tip>

<Tip>Activa el tutorial de bienvenida si quieres un repaso sobre cómo usar las funciones de Lasius</Tip>

<Tip>Toda la configuración se guarda localmente y persistirá entre sesiones pero no entre dispositivos</Tip>
