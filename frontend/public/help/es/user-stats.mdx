# Estadísticas Personales

Analiza tus datos de seguimiento de tiempo con gráficos y reportes visuales.

## Qué Puedes Hacer

### Cambiar Entre Vistas
Alterna entre dos modos de análisis:
- **Vista de Proyectos** - Ve la distribución de tiempo en todos tus proyectos, compara horas por proyecto, identifica dónde pasas la mayor parte del tiempo
- **Vista de Etiquetas** - Analiza el tiempo por categorías de etiqueta (ej. "Facturable", "Desarrollo", "Reuniones"), útil para facturación y planificación de recursos

### Explorar Gráficos
Interactúa con las representaciones visuales de datos:
- Pasa el cursor sobre elementos del gráfico para ver información detallada
- Ve gráficos de barras, gráficos circulares, gráficos de flujo y tablas detalladas
- Realiza un seguimiento de tendencias y patrones a lo largo del tiempo

### Filtrar Período de Tiempo
Usa el panel de filtros a la derechas para controlar el alcance de los datos:
- Selecciona períodos predefinidos: Hoy, Ayer, Esta semana, Última semana, Este mes, Último mes
- Establece rangos de fecha/hora personalizados con los campos **Desde** y **Hasta**
- Haz clic en **Restablecer** para limpiar todos los filtros

### Exportar Estadísticas
Haz clic en el botón **Exportar** para descargar tus datos:
- Formato CSV para aplicaciones de hojas de cálculo
- Formato Excel (.xlsx) para análisis adicional
- Incluye todos los datos filtrados y resúmenes
