# Estadísticas Personales

Analice sus datos de seguimiento de tiempo con gráficos y reportes visuales.

## Qué Puede Hacer

### Cambiar Entre Vistas
Alterne entre dos modos de análisis:
- **Vista de Proyectos** - Vea la distribución de tiempo en todos sus proyectos, compare horas por proyecto, identifique dónde pasa la mayor parte del tiempo
- **Vista de Etiquetas** - Analice el tiempo por categorías de etiqueta (ej. "Facturable", "Desarrollo", "Reuniones"), útil para facturación y planificación de recursos

### Explorar Gráficos
Interactúe con las representaciones visuales de datos:
- Pase el cursor sobre elementos del gráfico para ver información detallada
- Vea gráficos de barras, gráficos circulares, gráficos de flujo y tablas detalladas
- Realice un seguimiento de tendencias y patrones a lo largo del tiempo

### Filtrar Período de Tiempo
Use el panel de filtros a la derecha para controlar el alcance de los datos:
- Seleccione períodos predefinidos: Hoy, Ayer, Esta semana, Última semana, Este mes, Último mes
- Establezca rangos de fecha/hora personalizados con los campos **Desde** y **Hasta**
- Haga clic en **Restablecer** para limpiar todos los filtros

### Exportar Estadísticas
Haga clic en el botón **Exportar** para descargar sus datos:
- Formato CSV para aplicaciones de hojas de cálculo
- Formato Excel (.xlsx) para análisis adicional
- Incluye todos los datos filtrados y resúmenes
