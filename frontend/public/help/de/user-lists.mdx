# Buchungshistorie

Zeige, filtere und exportiere alle deine Zeitbuchungen in einem Tabellenformat.

## Was du tun kannst

### Buchungen anzeigen
Deine Buchungshistorie wird in einer Tabelle angezeigt, die Projekt, Tags, Datum/Zeit und Dauer zeigt.

### Buchungen filtern
Verwende das Filterpanel auf der rechten Seite, um deine Ansicht einzugrenzen:
- Wähle bestimmte Projekte aus
- Filtere nach Tags
- Wähle einen Zeitbereich (heute, diese Woche, diesen Monat oder benutzerdefinierte Daten)
- Klicke auf **Zurücksetzen**, um alle Filter zu löschen

### Auf Tags klicken
Klicke auf einen Tag-Chip in der Tabelle, um sofort nach diesem Tag zu filtern.

### Buchungsaktionen
Klicke auf das Kontextmenü (<Icon name="EllipsisVertical" />) bei jeder Buchung, um:
- **Buchung starten** - Beginne die Zeiterfassung mit demselben Projekt und denselben Tags
- **Buchung bearbeiten** - Ändere Start-/Endzeit, Projekt oder Tags
- **Start an vorherige Buchung anpassen** - Richte die Startzeit an der Endzeit der vorherigen Buchung aus
- **Zu Favoriten hinzufügen** - Speichere dieses Projekt für schnellen Zugriff
- **Buchung löschen** - Entferne diesen Eintrag

### Daten exportieren
Klicke auf die Schaltfläche **Exportieren** oben, um deine Buchungen als CSV- oder Excel-Dateien herunterzuladen.
