# Konto-Sicherheit

Aktualisiere dein Passwort, um dein Konto sicher zu halten.

## Was du tun kannst

### Passwort ändern
Aktualisiere dein Kontopasswort mit dem Formular:
1. Gib dein aktuelles **Passwort** ein
2. Gib dein **Neues Passwort** ein
3. **Neues Passwort bestätigen**, indem du es erneut eingibst
4. Klicke auf **Passwörter anzeigen** (<Icon name="Eye" />), um Passwortfelder bei Bedarf anzuzeigen
5. Klicke auf **Änderungen speichern**, um das neue Passwort anzuwenden

## Passwortanforderungen

Dein Passwort sollte:
- Mindestens 8 Zeichen lang sein
- Mindestens einen Großbuchstaben enthalten
- Mindestens eine Zahl enthalten
