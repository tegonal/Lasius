# Kontoeinstellungen

Verwalte deine persönlichen Informationen und Kontodetails.

## Was du tun kannst

### Profilinformationen aktualisieren
Bearbeite deine Kontodetails im Formular:
- **Rolle** - Deine Rolle wird von deinem Organisationsadministrator festgelegt (schreibgeschützt)
- **Vorname** - Dein Vorname
- **Nachname** - Dein Nachname
- **E-Mail** - Deine E-Mail-Adresse (kann nur geändert werden, wenn interne Authentifizierung verwendet wird)

Klicke auf **Änderungen speichern**, um deine Aktualisierungen anzuwenden.

## Tipps

<Tip>Deine E-Mail-Adresse kann nur geändert werden, wenn du den internen Authentifizierungsanbieter verwendest</Tip>

<Tip>E-Mail-Änderungen können eine Bestätigung erfordern</Tip>

<Tip>Alle Änderungen werden sofort nach dem Klicken auf Speichern gespeichert</Tip>
