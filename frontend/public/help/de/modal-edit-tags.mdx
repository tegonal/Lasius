# Projekt-Tags bearbeiten

Verwalte Tags und Tag-Gruppen für dein Projekt, um Buchungen für eine bessere Berichterstattung zu kategorisieren und zu filtern.

## Was du tun kannst

### Tag-Gruppen erstellen
Tag-Gruppen bündeln mehrere Tags für eine schnelle Anwendung während der Buchung:
1. Gib einen Namen in das Feld **Name** ein
2. Klicke auf **Tag-Gruppe erstellen**
3. Die neue Gruppe erscheint unten mit einem Eingabefeld
4. Gib Tag-Namen ein und drücke Enter, um Tags zur Gruppe hinzuzufügen
5. Klicke auf das Löschsymbol (<Icon name="Trash2" />), um die gesamte Gruppe zu löschen

### Einfache Tags hinzufügen
Füge einzelne Tags hinzu, die unabhängig verwendet werden können:
1. Klicke in das Feld **Tag eingeben, um es hinzuzufügen**
2. Gib den Tag-Namen ein und drücke Enter
3. Der Tag erscheint über dem Eingabefeld
4. Klicke auf das Entfernen-Symbol (<Icon name="X" />) bei einem Tag, um ihn zu entfernen

### Standard-Tag-Gruppen hinzufügen
Klicke auf **Standard-Tag-Gruppen hinzufügen**, um automatisch einen Standardsatz von Tag-Gruppen zu erstellen, einschließlich Abrechnungsstatus und Arbeitstypen.

### Änderungen speichern
Klicke auf **Speichern**, um deine Tag-Änderungen auf das Projekt anzuwenden, oder auf **Abbrechen**, um Änderungen zu verwerfen.

## Tipps

<Tip>Tag-Gruppen stellen konsistentes Tagging über Teammitglieder hinweg sicher</Tip>

<Tip>Verwende einfache Tags für einmalige Kategorien, die keine Gruppierung benötigen</Tip>

<Tip>Tags ermöglichen detailliertes Filtern in den Ansichten Statistiken und Listen</Tip>
