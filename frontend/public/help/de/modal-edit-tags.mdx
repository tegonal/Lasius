# Projekt-Tags bearbeiten

Verwalte Tags und Tag-Gruppen für dein Projekt, um Buchungen für eine bessere Berichterstattung zu kategorisieren und zu filtern.

## Benutzeroberfläche im Überblick

Der Tag-Manager hat zwei Tabs:
- **Tag-Gruppen**: Organisierte Sammlungen verwandter Tags
- **Einfache Tags**: Einzelne Tags, die nicht Teil einer Gruppe sind

## Tab "Tag-Gruppen"

### Tag-Gruppe erstellen
1. Klicke auf **Tag-Gruppe hinzufügen** in der Symbolleiste
2. Gib einen Namen im Modal-Dialog ein
3. Klicke auf **Tag-Gruppe erstellen**
4. Die neue Gruppe erscheint in alphabetischer Reihenfolge und wird automatisch erweitert

### Tags zu einer Gruppe hinzufügen
1. Klicke auf die **+** Schaltfläche auf der rechten Seite einer erweiterten Tag-Gruppe
2. Gib den Tag-Namen im Modal-Dialog ein
3. Klicke auf **Hinzufügen** oder drücke Enter
4. Der Tag erscheint in der Gruppe

### Tag-Gruppen verwalten

**Gruppen erweitern/einklappen**
- Klicke auf den Gruppennamen oder das Chevron-Symbol (<Icon name="ChevronUp" /> / <Icon name="ChevronDown" />), um einzelne Gruppen umzuschalten
- Verwende die Schaltfläche **Alle erweitern** / **Alle einklappen** oben rechts, um alle Gruppen auf einmal umzuschalten

**Tags kopieren und einfügen**
1. Erweitere eine Tag-Gruppe
2. Klicke auf das Kopier-Symbol (<Icon name="Copy" />), um alle Tags aus dieser Gruppe zu kopieren
3. Das Einfügen-Symbol (<Icon name="Clipboard" />) erscheint bei anderen Gruppen
4. Klicke auf Einfügen, um diese Tags zu einer anderen Gruppe hinzuzufügen (Duplikate werden automatisch gefiltert)

**Tag-Gruppe löschen**
1. Fahre mit der Maus über eine Tag-Gruppe, um das Löschen-Symbol (<Icon name="Trash2" />) anzuzeigen
2. Klicke auf das Löschen-Symbol
3. Bestätige das Löschen im erscheinenden Dialog

**Einzelne Tags entfernen**
- Klicke auf das **X** bei einem Tag-Badge, um ihn aus der Gruppe zu entfernen

**Tag-Anzahl**
- Jede Gruppe zeigt ein Badge mit der Anzahl der enthaltenen Tags

### Standard-Tag-Gruppen hinzufügen
Klicke auf **Standard-Tag-Gruppen hinzufügen** in der Symbolleiste, um automatisch einen Standardsatz von Tag-Gruppen und einfachen Tags zu erstellen, einschließlich Abrechnungsstatus und Arbeitstypen.

## Tab "Einfache Tags"

Füge einzelne Tags hinzu, die unabhängig verwendet werden können:
1. Wechsle zum Tab **Einfache Tags**
2. Klicke auf die **+** Schaltfläche
3. Gib den Tag-Namen im Modal-Dialog ein
4. Klicke auf **Hinzufügen** oder drücke Enter
5. Klicke auf das **X** bei einem Tag-Badge, um ihn zu entfernen

## Änderungen speichern

- Klicke auf **Speichern**, um deine Tag-Änderungen auf das Projekt anzuwenden
- Klicke auf **Abbrechen**, um Änderungen zu verwerfen
- Wenn du ungespeicherte Änderungen hast, wirst du vor dem Abbrechen um Bestätigung gebeten

## Tipps

<Tip>Tag-Gruppen stellen konsistentes Tagging über Teammitglieder hinweg sicher</Tip>

<Tip>Tag-Gruppen werden automatisch alphabetisch sortiert</Tip>

<Tip>Verwende Kopieren/Einfügen, um Tag-Strukturen schnell über Gruppen hinweg zu duplizieren</Tip>

<Tip>Verwende einfache Tags für einmalige Kategorien, die keine Gruppierung benötigen</Tip>

<Tip>Tags ermöglichen detailliertes Filtern in den Ansichten Statistiken und Listen</Tip>
