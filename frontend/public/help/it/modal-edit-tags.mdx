# Modifica Etichette Progetto

Gestisca le etichette e i gruppi di etichette per il suo progetto per categorizzare e filtrare le prenotazioni per una migliore reportistica.

## Cosa Può Fare

### Creare Gruppi di Etichette
I gruppi di etichette raggruppano più etichette insieme per un'applicazione rapida durante la prenotazione:
1. Inserisca un nome nel campo **Nome**
2. Faccia clic su **Crea gruppo di etichette**
3. Il nuovo gruppo appare sotto con un campo di input
4. Digiti i nomi delle etichette e prema Invio per aggiungere etichette al gruppo
5. Faccia clic sull'icona elimina (<Icon name="Trash2" />) per eliminare l'intero gruppo

### Aggiungere Etichette Semplici
Aggiunga etichette individuali che possono essere utilizzate indipendentemente:
1. Faccia clic nel campo **Inserisci un'etichetta per aggiungerla**
2. Digiti il nome dell'etichetta e prema Invio
3. L'etichetta appare sopra il campo di input
4. Faccia clic sull'icona rimuovi (<Icon name="X" />) su qualsiasi etichetta per rimuoverla

### Aggiungere Gruppi di Etichette Predefiniti
Faccia clic su **Aggiungi gruppi di etichette predefiniti** per creare automaticamente un set standard di gruppi di etichette, inclusi stato di fatturazione e tipi di lavoro.

### Salvare le Modifiche
Faccia clic su **Salva** per applicare le modifiche alle etichette del progetto, oppure su **Annulla** per annullare le modifiche.

## Suggerimenti

<Tip>I gruppi di etichette garantiscono un'etichettatura coerente tra i membri del team</Tip>

<Tip>Utilizzi le etichette semplici per categorie occasionali che non necessitano di raggruppamento</Tip>

<Tip>Le etichette consentono un filtraggio dettagliato nelle viste Statistiche e Elenchi</Tip>
