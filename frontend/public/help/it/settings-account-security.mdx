# Sicurezza Account

Aggiorni la tua password per mantenere sicuro il tuo account.

## Cosa Puoi Fare

### Cambiare Password
Aggiorni la password del tuo account utilizzando il modulo:
1. Inserisci la tua **Password** attuale
2. Inserisci la tua **Nuova password**
3. **Conferma la nuova password** inserendola di nuovo
4. Fai clic su **Mostra password** (<Icon name="Eye" />) per rivelare i campi password se necessario
5. Fai clic su **Salva modifiche** per applicare la nuova password

## Requisiti della Password

La tua password deve:
- Essere lunga almeno 8 caratteri
- Contenere almeno una lettera maiuscola
- Contenere almeno un numero
