# Impostazioni dell'app

Personalizza la tua esperienza Lasius configurando le preferenze dell'interfaccia.

## Cosa puoi fare

### Scegli la tua lingua
Seleziona la tua lingua preferita per l'interfaccia dal menu a discesa

### Seleziona il tuo tema
Scegli un tema che si adatta alle tue preferenze

### Controlla il tutorial di onboarding
Attiva o disattiva il tutorial di onboarding

Clicca su **Salva impostazioni** per applicare tutte le tue modifiche. La pagina si ricaricherà per attivare le tue nuove preferenze.

## Suggerimenti

<Tip>Cambiare la lingua ricaricherà l'app per applicare le traduzioni ovunque</Tip>

<Tip>Il tema di sistema passa automaticamente tra chiaro e scuro in base alle impostazioni del tuo dispositivo</Tip>

<Tip>Attiva il tutorial di onboarding se vuoi un ripasso su come utilizzare le funzionalità di Lasius</Tip>

<Tip>Tutte le impostazioni vengono salvate localmente e persisteranno tra le sessioni ma non tra i dispositivi</Tip>
