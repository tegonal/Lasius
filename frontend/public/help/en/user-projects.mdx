# My Projects

View and manage all projects you're a member of in the current organization.

## What You Can Do

### Browse Your Projects
See all projects you have access to displayed in a table showing project name, your role, and last activity.

### Manage Projects (Administrators)
Click the context menu (<Icon name="EllipsisVertical" />) on any project where you're an administrator to:
- **Mitglieder verwalten** - Add or remove project members and manage their roles
- **Buchungen anzeigen** - View all bookings made on this project
- **Statistiken anzeigen** - View detailed time tracking reports and analytics
- **Projekt bearbeiten** - Update project name, description, and settings
- **Tags bearbeiten** - Manage tag groups and simple tags for this project
- **Projekt deaktivieren** - Archive the project (moves it to inactive status)
- **Dieses Projekt verlassen** - Remove yourself from the project

### Leave Projects (Members)
As a member (non-administrator), click the context menu (<Icon name="EllipsisVertical" />) to leave the project.

### Create New Projects
Click **Create a project** in the right panel (administrators only) to add new projects to your organization.

## Tips

<Tip>Only administrators see the full context menu with project management options</Tip>

<Tip>Use "Buchungen anzeigen" (Show bookings) to review all time entries recorded on a specific project</Tip>

<Tip>Projects are automatically filtered by the currently selected organization in the top bar</Tip>

<Tip>The "Last activity" column helps you identify which projects have been recently used for time tracking</Tip>
