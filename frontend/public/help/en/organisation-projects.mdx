# Organisation Projects

Manage all projects within your organisation.

## What You Can Do

### View All Projects
See all active and archived projects in your organisation in a table format.

### Create New Project
Click **Create a project** to create a new project with name, description, team members, and tags.

### Manage Projects
Click the context menu (<Icon name="EllipsisVertical" />) on any project to:
- **Edit project details** - Update name, description, and settings
- **Add or remove members** - Control project access
- **Manage project tags** - Configure tag groups and simple tags
- **Archive project** - Hide completed projects from active list
- **View statistics** - See time tracking reports for the project

### Use Tags for Organization
Create tag groups and simple tags to categorize work and improve reporting.

## Tips

<Tip>Archive old projects to keep your list clean</Tip>

<Tip>Use descriptive project names for easy identification</Tip>
