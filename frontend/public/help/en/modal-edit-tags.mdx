# Edit Project Tags

Manage tags and tag groups for your project to categorize and filter bookings for better reporting.

## Interface Overview

The tag manager has two tabs:
- **Tag Groups**: Organized collections of related tags
- **Simple Tags**: Individual tags not part of any group

## Tag Groups Tab

### Create a Tag Group
1. Click **Add tag group** in the toolbar
2. Enter a name in the modal dialog
3. Click **Create tag group**
4. The new group appears in alphabetical order and expands automatically

### Add Tags to a Group
1. Click the **+** button on the right side of an expanded tag group
2. Enter the tag name in the modal dialog
3. Click **Add** or press Enter
4. The tag appears in the group

### Manage Tag Groups

**Expand/Collapse Groups**
- Click the group name or chevron icon (<Icon name="ChevronUp" /> / <Icon name="ChevronDown" />) to toggle individual groups
- Use the **Expand all** / **Collapse all** button in the top-right to toggle all groups at once

**Copy and Paste Tags**
1. Expand a tag group
2. Click the copy icon (<Icon name="Copy" />) to copy all tags from that group
3. The paste icon (<Icon name="Clipboard" />) appears on other groups
4. Click paste to add those tags to another group (duplicates are automatically filtered)

**Delete a Tag Group**
1. Hover over a tag group to reveal the delete icon (<Icon name="Trash2" />)
2. Click the delete icon
3. Confirm deletion in the dialog that appears

**Remove Individual Tags**
- Click the **X** on any tag badge to remove it from the group

**Tag Count**
- Each group shows a badge with the number of tags it contains

### Add Default Tag Groups
Click **Add default tag groups** in the toolbar to automatically create a standard set of tag groups and simple tags including billing status and work types.

## Simple Tags Tab

Add individual tags that can be used independently:
1. Switch to the **Simple Tags** tab
2. Click the **+** button
3. Enter the tag name in the modal dialog
4. Click **Add** or press Enter
5. Click the **X** on any tag badge to remove it

## Save Changes

- Click **Save** to apply your tag changes to the project
- Click **Cancel** to discard changes
- If you have unsaved changes, you'll be prompted to confirm before canceling

## Tips

<Tip>Tag groups ensure consistent tagging across team members</Tip>

<Tip>Tag groups are automatically sorted alphabetically</Tip>

<Tip>Use copy/paste to quickly duplicate tag structures across groups</Tip>

<Tip>Use simple tags for one-off categories that don't need grouping</Tip>

<Tip>Tags enable detailed filtering in Statistics and Lists views</Tip>
