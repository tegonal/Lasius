# Edit Project Tags

Manage tags and tag groups for your project to categorize and filter bookings for better reporting.

## What You Can Do

### Create Tag Groups
Tag groups bundle multiple tags together for quick application during booking:
1. Enter a name in the **Name** field
2. Click **Create tag group**
3. The new group appears below with an input field
4. Type tag names and press Enter to add tags to the group
5. Click the delete icon (<Icon name="Trash2" />) to delete the entire group

### Add Simple Tags
Add individual tags that can be used independently:
1. Click in the **Enter a tag to add it** field
2. Type the tag name and press Enter
3. The tag appears above the input field
4. Click the remove icon (<Icon name="X" />) on any tag to remove it

### Add Default Tag Groups
Click **Add default tag groups** to automatically create a standard set of tag groups including billing status and work types.

### Save Changes
Click **Save** to apply your tag changes to the project, or **Cancel** to discard changes.

## Tips

<Tip>Tag groups ensure consistent tagging across team members</Tip>

<Tip>Use simple tags for one-off categories that don't need grouping</Tip>

<Tip>Tags enable detailed filtering in Statistics and Lists views</Tip>
