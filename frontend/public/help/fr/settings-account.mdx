# Paramètres du compte

Gérez vos informations personnelles et les détails de votre compte.

## Ce que vous pouvez faire

### Mettre à jour les informations du profil
Modifiez les détails de votre compte dans le formulaire :
- **Rôle** - Votre rôle est déterminé par l'administrateur de votre organisation (lecture seule)
- **Prénom** - Votre prénom
- **Nom** - Votre nom de famille
- **Email** - Votre adresse email (ne peut être modifié que si vous utilisez l'authentification interne)

Cliquez sur **Enregistrer les modifications** pour appliquer vos mises à jour.

## Conseils

<Tip>Votre adresse email ne peut être modifiée que si vous utilisez le fournisseur d'authentification interne</Tip>

<Tip>Les modifications d'email peuvent nécessiter une vérification</Tip>

<Tip>Toutes les modifications sont enregistrées immédiatement après avoir cliqué sur Enregistrer</Tip>
