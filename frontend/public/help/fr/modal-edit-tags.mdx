# Modifier les étiquettes de projet

Gérez les étiquettes et les groupes d'étiquettes pour votre projet afin de catégoriser et filtrer les réservations pour de meilleurs rapports.

## Ce que vous pouvez faire

### Créer des groupes d'étiquettes
Les groupes d'étiquettes regroupent plusieurs étiquettes pour une application rapide lors de la réservation :
1. Entrez un nom dans le champ **Nom**
2. Cliquez sur **Créer un groupe d'étiquettes**
3. Le nouveau groupe apparaît ci-dessous avec un champ de saisie
4. Tapez les noms d'étiquettes et appuyez sur Entrée pour ajouter des étiquettes au groupe
5. Cliquez sur l'icône de suppression (<Icon name="Trash2" />) pour supprimer l'ensemble du groupe

### Ajouter des étiquettes simples
Ajoutez des étiquettes individuelles qui peuvent être utilisées indépendamment :
1. Cliquez dans le champ **Entrez une étiquette pour l'ajouter**
2. Tapez le nom de l'étiquette et appuyez sur Entrée
3. L'étiquette apparaît au-dessus du champ de saisie
4. Cliquez sur l'icône de suppression (<Icon name="X" />) sur n'importe quelle étiquette pour la retirer

### Ajouter des groupes d'étiquettes par défaut
Cliquez sur **Ajouter des groupes d'étiquettes par défaut** pour créer automatiquement un ensemble standard de groupes d'étiquettes, y compris le statut de facturation et les types de travail.

### Enregistrer les modifications
Cliquez sur **Enregistrer** pour appliquer vos modifications d'étiquettes au projet, ou **Annuler** pour les abandonner.

## Conseils

<Tip>Les groupes d'étiquettes garantissent un étiquetage cohérent entre les membres de l'équipe</Tip>

<Tip>Utilisez des étiquettes simples pour des catégories ponctuelles qui ne nécessitent pas de regroupement</Tip>

<Tip>Les étiquettes permettent un filtrage détaillé dans les vues Statistiques et Listes</Tip>
