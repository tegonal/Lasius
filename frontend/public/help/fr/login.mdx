# Connexion

## Méthodes de connexion

Choisis parmi les méthodes d'authentification disponibles configurées par ton administrateur. Les options que tu vois dépendent de la configuration de ton organisation et peuvent inclure :

- Authentification interne Lasius
- Fournisseurs d'authentification unique (SSO)
- Autres fournisseurs d'identité

Clique sur le bouton de ta méthode de connexion préférée pour continuer.

## Pas de compte ?

Contacte l'administrateur de ton organisation pour recevoir une invitation.
