# Connexion

## Méthodes de connexion

Choisissez parmi les méthodes d'authentification disponibles configurées par votre administrateur. Les options que vous voyez dépendent de la configuration de votre organisation et peuvent inclure :

- Authentification interne Lasius
- Fournisseurs d'authentification unique (SSO)
- Autres fournisseurs d'identité

Cliquez sur le bouton de votre méthode de connexion préférée pour continuer.

## Pas de compte ?

Contactez l'administrateur de votre organisation pour recevoir une invitation.
