# Statistiques personnelles

Analysez vos données de suivi du temps avec des graphiques visuels et des rapports.

## Ce que vous pouvez faire

### Basculer entre les vues
Basculez entre deux modes d'analyse :
- **Vue Projets** - Voir la répartition du temps sur tous vos projets, comparer les heures par projet, identifier où vous passez le plus de temps
- **Vue Étiquettes** - Analyser le temps par catégories d'étiquettes (par ex. "Facturable", "Développement", "Réunions"), utile pour la facturation et la planification des ressources

### Explorer les graphiques
Interagissez avec les représentations visuelles des données :
- Survolez les éléments du graphique pour voir des informations détaillées
- Consultez des graphiques à barres, des graphiques circulaires, des graphiques en flux et des tableaux détaillés
- Suivez les tendances et les modèles dans le temps

### Filtrer la période
Utilisez le panneau de filtrage à droite pour contrôler la portée des données :
- Sélectionnez des périodes prédéfinies : Aujourd'hui, Hier, Cette semaine, La semaine dernière, Ce mois, Le mois dernier
- Définissez des plages de date/heure personnalisées avec les champs **De** et **À**
- Cliquez sur **Réinitialiser** pour effacer tous les filtres

### Exporter les statistiques
Cliquez sur le bouton **Exporter** pour télécharger vos données :
- Format CSV pour les applications de tableur
- Format Excel (.xlsx) pour une analyse plus approfondie
- Inclut toutes les données filtrées et les résumés

## Conseils

<Tip>Utilisez la vue Projets pour la facturation client et la budgétisation de projet</Tip>

<Tip>Utilisez la vue Étiquettes pour comprendre comment vous catégorisez votre travail</Tip>

<Tip>Comparez différentes périodes pour identifier les tendances</Tip>

<Tip>Exportez les données mensuellement pour les rapports de gestion</Tip>

<Tip>Examinez les statistiques hebdomadairement pour assurer une allocation de projet équilibrée</Tip>

<Tip>Utilisez des plages de dates personnalisées pour des revues trimestrielles ou annuelles</Tip>
