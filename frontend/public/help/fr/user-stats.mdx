# Statistiques personnelles

Analyse tes données de suivi du temps avec des graphiques visuels et des rapports.

## Ce que tu peux faire

### Basculer entre les vues
Bascule entre deux modes d'analyse :
- **Vue Projets** - Voir la répartition du temps sur tous tes projets, comparer les heures par projet, identifier où tu passes le plus de temps
- **Vue Étiquettes** - Analyser le temps par catégories d'étiquettes (par ex. "Facturable", "Développement", "Réunions"), utile pour la facturation et la planification des ressources

### Explorer les graphiques
Interagis avec les représentations visuelles des données :
- Survole les éléments du graphique pour voir des informations détaillées
- Consulte des graphiques à barres, des graphiques circulaires, des graphiques en flux et des tableaux détaillés
- Suis les tendances et les modèles dans le temps

### Filtrer la période
Utilise le panneau de filtrage à droite pour contrôler la portée des données :
- Sélectionne des périodes prédéfinies : Aujourd'hui, Hier, Cette semaine, La semaine dernière, Ce mois, Le mois dernier
- Définis des plages de date/heure personnalisées avec les champs **De** et **À**
- Clique sur **Réinitialiser** pour effacer tous les filtres

### Exporter les statistiques
Clique sur le bouton **Exporter** pour télécharger tes données :
- Format CSV pour les applications de tableur
- Format Excel (.xlsx) pour une analyse plus approfondie
- Inclut toutes les données filtrées et les résumés

## Conseils

<Tip>Utilise la vue Projets pour la facturation client et la budgétisation de projet</Tip>

<Tip>Utilise la vue Étiquettes pour comprendre comment tu catégorises ton travail</Tip>

<Tip>Compare différentes périodes pour identifier les tendances</Tip>

<Tip>Exporte les données mensuellement pour les rapports de gestion</Tip>

<Tip>Examine les statistiques hebdomadairement pour assurer une allocation de projet équilibrée</Tip>

<Tip>Utilise des plages de dates personnalisées pour des revues trimestrielles ou annuelles</Tip>
