# Projets de l'organisation

Gère tous les projets au sein de ton organisation. Cette vue est disponible pour les administrateurs de l'organisation et fournit une supervision de tous les projets, y compris ceux où des membres externes peuvent réserver du temps.

## Ce que tu peux faire

### Voir l'aperçu des projets
En haut de la page, consulte les statistiques montrant les projets totaux, actifs et inactifs dans ton organisation.

### Filtrer les projets par statut
Utilise les boutons de filtre de statut pour afficher :
- **Les deux** - Tous les projets quel que soit leur statut
- **Actifs** - Seulement les projets actifs
- **Inactifs** - Seulement les projets archivés/désactivés

### Parcourir tous les projets
Consulte tous les projets de l'organisation dans un tableau montrant le nom du projet, le statut et la dernière activité.

### Créer de nouveaux projets
Clique sur **Créer un projet** dans le panneau de droite pour créer un nouveau projet avec nom, description, membres de l'équipe et étiquettes.

### Gérer les projets
Clique sur le menu contextuel (<Icon name="EllipsisVertical" />) sur n'importe quel projet :
- **Gérer les membres** - Ajouter ou retirer des membres du projet, y compris des membres externes de l'extérieur de ton organisation
- **Afficher les réservations** - Voir toutes les réservations effectuées sur ce projet par tous les membres
- **Afficher les statistiques** - Consulter les rapports et analyses complets de suivi du temps
- **Modifier le projet** - Mettre à jour le nom, la description et les paramètres du projet
- **Modifier les étiquettes** - Gérer les groupes d'étiquettes et les étiquettes simples pour catégoriser le travail
- **Désactiver le projet** - Archiver le projet (ou le réactiver s'il est actuellement inactif)

### Créer des rapports de facturation
Génère des rapports incluant le temps réservé par des membres de projet externes - des personnes extérieures à ton organisation qui ont obtenu l'accès à des projets spécifiques.

## Conseils

<Tip>Utilise le filtre de statut pour te concentrer sur les projets actifs ou examiner séparément les projets archivés</Tip>

<Tip>Les statistiques en haut te donnent un aperçu rapide du nombre de projets que ton organisation gère</Tip>

<Tip>Les membres de projet externes peuvent réserver du temps même s'ils ne font pas partie de ton organisation - parfait pour les sous-traitants ou partenaires</Tip>

<Tip>Archive les projets terminés pour garder ta liste de projets actifs propre et organisée</Tip>

<Tip>La colonne "Dernière activité" aide à identifier les projets activement utilisés vs. les projets dormants qui pourraient nécessiter un archivage</Tip>
