# Projets de l'organisation

Gérez tous les projets au sein de votre organisation. Cette vue est disponible pour les administrateurs de l'organisation et fournit une supervision de tous les projets, y compris ceux où des membres externes peuvent réserver du temps.

## Ce que vous pouvez faire

### Voir l'aperçu des projets
En haut de la page, consultez les statistiques montrant les projets totaux, actifs et inactifs dans votre organisation.

### Filtrer les projets par statut
Utilisez les boutons de filtre de statut pour afficher :
- **Beide** (Les deux) - Tous les projets quel que soit leur statut
- **Aktiv** - Seulement les projets actifs
- **Inaktiv** - Seulement les projets archivés/désactivés

### Parcourir tous les projets
Consultez tous les projets de l'organisation dans un tableau montrant le nom du projet, le statut et la dernière activité.

### Créer de nouveaux projets
Cliquez sur **Créer un projet** dans le panneau de droite pour créer un nouveau projet avec nom, description, membres de l'équipe et étiquettes.

### Gérer les projets
Cliquez sur le menu contextuel (<Icon name="EllipsisVertical" />) sur n'importe quel projet :
- **Mitglieder verwalten** - Ajouter ou retirer des membres du projet, y compris des membres externes de l'extérieur de votre organisation
- **Buchungen anzeigen** - Voir toutes les réservations effectuées sur ce projet par tous les membres
- **Statistiken anzeigen** - Consulter les rapports et analyses complets de suivi du temps
- **Projekt bearbeiten** - Mettre à jour le nom, la description et les paramètres du projet
- **Tags bearbeiten** - Gérer les groupes d'étiquettes et les étiquettes simples pour catégoriser le travail
- **Projekt deaktivieren** - Archiver le projet (ou le réactiver s'il est actuellement inactif)

### Créer des rapports de facturation
Générez des rapports incluant le temps réservé par des membres de projet externes - des personnes extérieures à votre organisation qui ont obtenu l'accès à des projets spécifiques.

## Conseils

<Tip>Utilisez le filtre de statut pour vous concentrer sur les projets actifs ou examiner séparément les projets archivés</Tip>

<Tip>Les statistiques en haut vous donnent un aperçu rapide du nombre de projets que votre organisation gère</Tip>

<Tip>Les membres de projet externes peuvent réserver du temps même s'ils ne font pas partie de votre organisation - parfait pour les sous-traitants ou partenaires</Tip>

<Tip>Archivez les projets terminés pour garder votre liste de projets actifs propre et organisée</Tip>

<Tip>La colonne "Dernière activité" aide à identifier les projets activement utilisés vs. les projets dormants qui pourraient nécessiter un archivage</Tip>
