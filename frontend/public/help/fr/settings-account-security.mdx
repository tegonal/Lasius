# Sécurité du compte

Mettez à jour votre mot de passe pour sécuriser votre compte.

## Ce que vous pouvez faire

### Changer le mot de passe
Mettez à jour le mot de passe de votre compte à l'aide du formulaire :
1. Entrez votre **Mot de passe** actuel
2. Entrez votre **Nouveau mot de passe**
3. **Confirmez le nouveau mot de passe** en le saisissant à nouveau
4. Cliquez sur **Afficher les mots de passe** (<Icon name="Eye" />) pour révéler les champs de mot de passe si nécessaire
5. Cliquez sur **Enregistrer les modifications** pour appliquer le nouveau mot de passe

## Exigences du mot de passe

Votre mot de passe doit :
- Avoir au moins 8 caractères
- Contenir au moins une lettre majuscule
- Contenir au moins un chiffre

## Conseils
