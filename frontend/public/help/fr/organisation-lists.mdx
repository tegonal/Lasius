# Historique des réservations de l'organisation

Consulte et analyse toutes les réservations de temps de ton organisation entière.

## Ce que tu peux faire

### Voir les réservations de l'équipe
Consulte les entrées de temps de tous les membres de l'équipe de ton organisation affichées dans un tableau avec le projet, l'utilisateur, les étiquettes, la date/heure et la durée.

### Filtrer les réservations
Utilise le panneau de filtrage à droite pour affiner ta vue :
- Sélectionne des membres spécifiques de l'équipe
- Sélectionne des projets spécifiques
- Filtre par étiquettes
- Choisis une période (aujourd'hui, cette semaine, ce mois, ou dates personnalisées)
- Clique sur **Réinitialiser** pour effacer tous les filtres

### Exporter les données
Clique sur le bouton **Exporter** en haut pour télécharger tes réservations sous forme de fichiers CSV ou Excel.

### Analyser l'activité de l'équipe
Examine la répartition du temps entre les projets et les membres de l'équipe pour identifier les modèles et les goulots d'étranglement.

## Conseils

<Tip>Utilise les filtres pour générer des rapports spécifiques aux clients</Tip>

<Tip>Exporte régulièrement les données pour la sauvegarde et l'analyse</Tip>

<Tip>Examine les réservations de l'équipe pour identifier les goulots d'étranglement</Tip>
