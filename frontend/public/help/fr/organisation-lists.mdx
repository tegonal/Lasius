# Historique des réservations de l'organisation

Consultez et analysez toutes les réservations de temps de votre organisation entière.

## Ce que vous pouvez faire

### Voir les réservations de l'équipe
Consultez les entrées de temps de tous les membres de l'équipe de votre organisation affichées dans un tableau avec le projet, l'utilisateur, les étiquettes, la date/heure et la durée.

### Filtrer les réservations
Utilisez le panneau de filtrage à droite pour affiner votre vue :
- Sélectionnez des membres spécifiques de l'équipe
- Sélectionnez des projets spécifiques
- Filtrez par étiquettes
- Choisissez une période (aujourd'hui, cette semaine, ce mois, ou dates personnalisées)
- Cliquez sur **Réinitialiser** pour effacer tous les filtres

### Exporter les données
Cliquez sur le bouton **Exporter** en haut pour télécharger vos réservations sous forme de fichiers CSV ou Excel.

### Analyser l'activité de l'équipe
Examinez la répartition du temps entre les projets et les membres de l'équipe pour identifier les modèles et les goulots d'étranglement.

## Conseils

<Tip>Utilisez les filtres pour générer des rapports spécifiques aux clients</Tip>

<Tip>Exportez régulièrement les données pour la sauvegarde et l'analyse</Tip>

<Tip>Examinez les réservations de l'équipe pour identifier les goulots d'étranglement</Tip>
