# Réservations - Page d'accueil

Suivez votre temps et gérez vos réservations quotidiennes.

## Ce que vous pouvez faire

### Naviguer entre les jours
Utilisez le sélecteur de date en haut pour afficher les réservations de différents jours. Cliquez sur les flèches pour naviguer ou cliquez sur un bouton de jour spécifique. Chaque bouton de jour affiche une barre de progression indiquant combien de votre journée de travail attendue vous avez enregistré.

### Changer d'organisation
Utilisez le sélecteur d'organisation dans la barre supérieure si vous êtes membre de plusieurs équipes.

### Afficher la réservation en cours
Lors du suivi du temps, la barre d'état affiche le nom du projet actif, la durée en cours et un bouton d'arrêt (<Icon name="Square" />) pour terminer le suivi.

### Gérer les réservations du jour
Chaque entrée de réservation affiche le projet, les étiquettes, l'heure de début/fin et la durée. Entre les réservations, des flèches indicatrices montrent les écarts ou les chevauchements dans votre suivi du temps.

Cliquez sur le menu contextuel (<Icon name="EllipsisVertical" />) sur n'importe quelle réservation pour :
- **Démarrer la réservation** - Commencer à suivre le temps avec le même projet et les mêmes étiquettes
- **Modifier la réservation** - Modifier l'heure de début, l'heure de fin, le projet ou les étiquettes
- **Ajuster le début à la réservation précédente** - Aligner automatiquement l'heure de début avec la fin de la réservation précédente
- **Ajouter aux favoris** - Marquer ce projet pour un accès rapide
- **Supprimer la réservation** - Retirer cette entrée de temps

### Insérer des réservations
Utilisez les boutons **Ajouter une réservation** et **Insérer une réservation** entre les entrées pour combler les lacunes dans votre chronologie.

### Commencer à suivre le temps
Le panneau de droite offre trois façons de commencer le suivi :

**Onglet 1 : Démarrer une réservation** (<Icon name="Clock" />)
- Sélectionnez un projet dans le menu déroulant (obligatoire)
- Ajoutez des étiquettes optionnelles pour catégoriser votre travail
- Cliquez sur **Démarrer la réservation** (<Icon name="PlayCircle" />) pour commencer le suivi immédiatement
- Ou cliquez sur **Créer une réservation** pour ajouter une entrée manuelle avec des heures spécifiques

**Onglet 2 : Démarrer depuis un favori** (<Icon name="Star" />)
- Cliquez sur n'importe quelle réservation favorite pour commencer rapidement le suivi avec cette combinaison de projet et d'étiquettes
- Utilisez le menu contextuel (<Icon name="EllipsisVertical" />) pour des actions supplémentaires

**Onglet 3 : Démarrer depuis un membre de l'équipe** (<Icon name="Users" />)
- Cliquez sur l'avatar d'un membre de l'équipe pour voir ses réservations récentes
- Copiez leurs réservations pour suivre le temps sur les mêmes projets

## Conseils

<Tip>Le sélecteur de date en haut vous permet de consulter et modifier les réservations passées</Tip>

<Tip>Utilisez les étiquettes de manière cohérente pour générer des rapports significatifs</Tip>

<Tip>Le menu contextuel (<Icon name="EllipsisVertical" />) sur chaque réservation offre des actions rapides</Tip>

<Tip>Utilisez "Ajouter aux favoris" dans le menu contextuel pour enregistrer les combinaisons de projet et d'étiquettes fréquemment utilisées pour un accès rapide</Tip>

<Tip>L'onglet "Démarrer depuis un membre de l'équipe" facilite la visualisation de ce sur quoi vos collègues travaillent</Tip>
