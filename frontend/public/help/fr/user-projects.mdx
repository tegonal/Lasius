# Mes projets

Consultez et gérez tous les projets dont vous êtes membre dans l'organisation actuelle.

## Ce que vous pouvez faire

### Parcourir vos projets
Consultez tous les projets auxquels vous avez accès affichés dans un tableau montrant le nom du projet, votre rôle et la dernière activité.

### Gérer les projets (administrateurs)
Cliquez sur le menu contextuel (<Icon name="EllipsisVertical" />) sur n'importe quel projet dont vous êtes administrateur :
- **Mitglieder verwalten** - Ajouter ou retirer des membres du projet et gérer leurs rôles
- **Buchungen anzeigen** - Voir toutes les réservations effectuées sur ce projet
- **Statistiken anzeigen** - Consulter les rapports et analyses détaillés de suivi du temps
- **Projekt bearbeiten** - Mettre à jour le nom, la description et les paramètres du projet
- **Tags bearbeiten** - Gérer les groupes d'étiquettes et les étiquettes simples pour ce projet
- **Projekt deaktivieren** - Archiver le projet (le déplacer vers un statut inactif)
- **Dieses Projekt verlassen** - Vous retirer du projet

### Quitter les projets (membres)
En tant que membre (non-administrateur), cliquez sur le menu contextuel (<Icon name="EllipsisVertical" />) pour quitter le projet.

### Créer de nouveaux projets
Cliquez sur **Créer un projet** dans le panneau de droite (administrateurs uniquement) pour ajouter de nouveaux projets à votre organisation.

## Conseils

<Tip>Seuls les administrateurs voient le menu contextuel complet avec les options de gestion de projet</Tip>

<Tip>Utilisez "Buchungen anzeigen" (Afficher les réservations) pour consulter toutes les entrées de temps enregistrées sur un projet spécifique</Tip>

<Tip>Les projets sont automatiquement filtrés par l'organisation actuellement sélectionnée dans la barre supérieure</Tip>

<Tip>La colonne "Dernière activité" vous aide à identifier les projets qui ont été récemment utilisés pour le suivi du temps</Tip>
