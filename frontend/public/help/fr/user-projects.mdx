# Mes projets

Consulte et gère tous les projets dont tu es membre dans l'organisation actuelle.

## Ce que tu peux faire

### Parcourir tes projets
Consulte tous les projets auxquels tu as accès affichés dans un tableau montrant le nom du projet, ton rôle et la dernière activité.

### Gérer les projets (administrateurs)
Clique sur le menu contextuel (<Icon name="EllipsisVertical" />) sur n'importe quel projet dont tu es administrateur :
- **Gérer les membres** - Ajouter ou retirer des membres du projet et gérer leurs rôles
- **Afficher les réservations** - Voir toutes les réservations effectuées sur ce projet
- **Afficher les statistiques** - Consulter les rapports et analyses détaillés de suivi du temps
- **Modifier le projet** - Mettre à jour le nom, la description et les paramètres du projet
- **Modifier les étiquettes** - Gérer les groupes d'étiquettes et les étiquettes simples pour ce projet
- **Désactiver le projet** - Archiver le projet (le déplacer vers un statut inactif)
- **Quitter ce projet** - Te retirer du projet

### Quitter les projets (membres)
En tant que membre (non-administrateur), clique sur le menu contextuel (<Icon name="EllipsisVertical" />) pour quitter le projet.

### Créer de nouveaux projets
Clique sur **Créer un projet** dans le panneau de droite (administrateurs uniquement) pour ajouter de nouveaux projets à ton organisation.

## Conseils

<Tip>Seuls les administrateurs voient le menu contextuel complet avec les options de gestion de projet</Tip>

<Tip>Utilise "Afficher les réservations" pour consulter toutes les entrées de temps enregistrées sur un projet spécifique</Tip>

<Tip>Les projets sont automatiquement filtrés par l'organisation actuellement sélectionnée dans la barre supérieure</Tip>

<Tip>La colonne "Dernière activité" t'aide à identifier les projets qui ont été récemment utilisés pour le suivi du temps</Tip>
