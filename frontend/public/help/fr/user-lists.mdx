# Historique des réservations

Consultez, filtrez et exportez toutes vos réservations de temps dans un format tableau.

## Ce que vous pouvez faire

### Voir les réservations
Votre historique de réservations est affiché dans un tableau montrant le projet, les étiquettes, la date/heure et la durée.

### Filtrer les réservations
Utilisez le panneau de filtrage à droite pour affiner votre vue :
- Sélectionnez des projets spécifiques
- Filtrez par étiquettes
- Choisissez une période (aujourd'hui, cette semaine, ce mois, ou dates personnalisées)
- Cliquez sur **Réinitialiser** pour effacer tous les filtres

### Cliquer sur les étiquettes
Cliquez sur n'importe quelle puce d'étiquette dans le tableau pour filtrer instantanément les réservations par cette étiquette.

### Actions de réservation
Cliquez sur le menu contextuel (<Icon name="EllipsisVertical" />) sur n'importe quelle réservation pour :
- **Démarrer la réservation** - Commencer à suivre le temps avec le même projet et les mêmes étiquettes
- **Modifier la réservation** - Modifier l'heure de début/fin, le projet ou les étiquettes
- **Ajuster le début à la réservation précédente** - Aligner l'heure de début avec la fin de la réservation précédente
- **Ajouter aux favoris** - Enregistrer ce projet pour un accès rapide
- **Supprimer la réservation** - Retirer cette entrée

### Exporter les données
Cliquez sur le bouton **Exporter** en haut pour télécharger vos réservations sous forme de fichiers CSV ou Excel.

## Conseils

<Tip>Cliquez sur les puces d'étiquettes dans le tableau pour filtrer rapidement par cette étiquette</Tip>

<Tip>Utilisez les périodes prédéfinies pour des vues rapides mensuelles ou hebdomadaires</Tip>

<Tip>Combinez les filtres de projet et d'étiquettes pour des rapports détaillés</Tip>

<Tip>Le menu contextuel fournit des options rapides de modification et de suppression</Tip>
