# Statistiques de l'organisation

Analyse les données de suivi du temps de ton organisation entière avec des rapports et des visualisations complets.

## Ce que tu peux faire

### Voir les métriques de l'organisation
Consulte des statistiques de haut niveau incluant le total d'heures suivies, les projets actifs, l'activité des membres de l'équipe et la répartition des projets.

### Analyser les performances de l'équipe
Examine le temps suivi par membre de l'équipe, les tendances de productivité, l'allocation des projets et la répartition de la charge de travail.

### Consulter les rapports de projet
Suis le temps par projet, surveille les progrès du projet, suis les budgets et analyse les chronologies.

### Basculer entre les vues
Bascule entre deux modes d'analyse :
- **Vue Projets** - Voir la répartition du temps sur tous les projets
- **Vue Étiquettes** - Analyser le temps par catégories d'étiquettes
- **Vue Utilisateurs** - Examiner le temps suivi par membre de l'équipe

### Filtrer la période
Utilise le panneau de filtrage à droite pour contrôler la portée des données :
- Sélectionne des périodes prédéfinies : Aujourd'hui, Hier, Cette semaine, La semaine dernière, Ce mois, Le mois dernier
- Définis des plages de date/heure personnalisées avec les champs **De** et **À**
- Clique sur **Réinitialiser** pour effacer tous les filtres

### Exporter les statistiques
Clique sur le bouton **Exporter** pour télécharger tes données :
- Format CSV pour les applications de tableur
- Format Excel (.xlsx) pour une analyse plus approfondie
- Inclut toutes les données filtrées et les résumés
