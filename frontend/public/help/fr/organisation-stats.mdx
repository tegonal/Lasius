# Statistiques de l'organisation

Analysez les données de suivi du temps de votre organisation entière avec des rapports et des visualisations complets.

## Ce que vous pouvez faire

### Voir les métriques de l'organisation
Consultez des statistiques de haut niveau incluant le total d'heures suivies, les projets actifs, l'activité des membres de l'équipe et la répartition des projets.

### Analyser les performances de l'équipe
Examinez le temps suivi par membre de l'équipe, les tendances de productivité, l'allocation des projets et la répartition de la charge de travail.

### Consulter les rapports de projet
Suivez le temps par projet, surveillez les progrès du projet, suivez les budgets et analysez les chronologies.

### Basculer entre les vues
Basculez entre deux modes d'analyse :
- **Vue Projets** - Voir la répartition du temps sur tous les projets
- **Vue Étiquettes** - Analyser le temps par catégories d'étiquettes
- **Vue Utilisateurs** - Examiner le temps suivi par membre de l'équipe

### Filtrer la période
Utilisez le panneau de filtrage à droite pour contrôler la portée des données :
- Sélectionnez des périodes prédéfinies : Aujourd'hui, Hier, Cette semaine, La semaine dernière, Ce mois, Le mois dernier
- Définissez des plages de date/heure personnalisées avec les champs **De** et **À**
- Cliquez sur **Réinitialiser** pour effacer tous les filtres

### Exporter les statistiques
Cliquez sur le bouton **Exporter** pour télécharger vos données :
- Format CSV pour les applications de tableur
- Format Excel (.xlsx) pour une analyse plus approfondie
- Inclut toutes les données filtrées et les résumés

## Conseils
