# Organisation

Consultez et gérez les détails de votre organisation et les membres de l'équipe.

## Aperçu de l'organisation

En haut de la page, vous verrez :
- **Nom de l'organisation** - Le nom de votre organisation actuelle
- **Nombre de membres** - Nombre total de membres de l'équipe
- **Menu Actions** - Accès rapide à la gestion de l'organisation (administrateurs uniquement)

## Liste des membres

Tous les membres de l'organisation sont affichés dans un tableau montrant :
- **Avatar** - Identifiant visuel pour chaque membre
- **Nom** - Nom complet du membre
- **Email** - Adresse email du membre
- **Statut** - Affiche le badge "Vous" pour votre propre compte

### Gestion des membres (administrateurs uniquement)

**Inviter de nouveaux membres**
1. Cliquez sur le bouton **Actions** en haut à droite
2. Sélectionnez **Inviter quelqu'un**
3. Entrez l'adresse email de la personne à inviter
4. Envoyez l'invitation

**Retirer des membres**
1. Cliquez sur le menu contextuel (<Icon name="EllipsisVertical" />) sur n'importe quelle ligne de membre
2. Sélectionnez **Retirer le membre**
3. Confirmez l'action de suppression

<Note>Vous ne pouvez pas vous retirer vous-même d'une organisation. Les membres sont triés alphabétiquement par nom de famille.</Note>

## Paramètres de l'organisation (administrateurs uniquement)

Utilisez le menu **Actions** pour :
- **Inviter quelqu'un** - Envoyer des invitations par email à de nouveaux membres de l'équipe
- **Modifier l'organisation** - Modifier le nom et les paramètres de l'organisation
- **Créer une organisation** - Configurer une nouvelle organisation pour gérer des équipes supplémentaires

<Warning>Retirer un membre supprime immédiatement son accès à tous les projets de l'organisation.</Warning>

## Conseils

<Tip>Gardez le nom de votre organisation clair et reconnaissable pour tous les membres</Tip>

<Tip>Examinez régulièrement votre liste de membres pour assurer un contrôle d'accès approprié</Tip>

<Tip>Utilisez le menu contextuel sur chaque membre pour gérer rapidement son adhésion</Tip>
